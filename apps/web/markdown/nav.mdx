# Nav

Navs are styled [`<nav>`](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/nav) HTML elements.

## Example

<nav className="nav">
  <a className="link-text" href="#">Influx UI</a>
</nav>
<br />

```jsx
<nav class="nav">
  <a class="link-text" href="#">Influx UI</a>
</nav>
```

## Alignment

### Space Between Nav

<nav className="nav nav-between">
  <a className="link-text" href="#">Influx UI</a>
  <button className="button">Sign Up</button>
</nav>
<br />

```jsx
<nav class="nav nav-between">
  <a class="link-text" href="#">Influx UI</a>
  <button class="button">Sign Up</button>
</nav>
```

## Centered Nav

<nav className="nav nav-center">
  <a className="link-text" href="#">Influx UI</a>
</nav>
<br />

```jsx
<nav class="nav nav-center">
  <a class="link-text" href="#">Influx UI</a>
</nav>
```
