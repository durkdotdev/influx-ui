# Form

Forms are styled [`<form>`](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/form) HTML elements.

<form className="form">
  <label className="label">
    <span>First Name</span>
    <input className="input" placeholder="First Name" />
  </label>
  <button className="button">Submit</button>
</form>
<br />

```jsx
<form className="form">
  <label className="label">
    <span>First Name</span>
    <input className="input" placeholder="First Name" />
  </label>
  <button className="button">Submit</button>
</form>
```
