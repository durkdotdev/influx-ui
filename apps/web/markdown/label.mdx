# Label

Labels are styled [`<label>`](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/label) HTML elements. Wrap inputs with them to provide a label.

<label className="label">
  <span>First Name</span>
  <input className="input" placeholder="First Name" />
</label>
<br />

```jsx
<label className="label">
  <span>First Name</span>
  <input className="input" placeholder="First Name" />
</label>
```
