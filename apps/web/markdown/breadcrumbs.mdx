# Breadcrumbs

Breadcrumbs are a great way to show users how to find their way through a website.

## Example

<ul className="breadcrumbs">
  <li>
    <a href="#">Home</a>
  </li>
  <li>
    <a href="#">Account</a>
  </li>
  <li>Settings</li>
</ul>
<br />

```jsx
<ul className="breadcrumbs">
  <li>
    <a href="#">Home</a>
  </li>
  <li>
    <a href="#">Account</a>
  </li>
  <li>Settings</li>
</ul>
```
