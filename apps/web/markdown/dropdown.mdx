# Dropdown

Dropdowns can display content relative to a specific HTML element.

## Example

<DropdownDemo />
<br />

```jsx
<div className="dropdown-container">
  <div className="dropdown">Dropdown content...</div>
</div>
```

## React Example

```jsx
// dropdown.tsx

import { useState } from "react";

const DropdownDemo = () => {
  const [isShown, setIsShown] = useState < boolean > false;

  return (
    <div className="dropdown-container">
      <button
        className="button button-light"
        onClick={() => setIsShown(!isShown)}
      >
        {isShown ? "Close " : "Open "}Dropdown
      </button>

      {isShown && <div className="dropdown">Dropdown content...</div>}
    </div>
  );
};

export default DropdownDemo;
```

## Close on Click Outside

You may use a package such as [`use-onlickoutside`](https://www.npmjs.com/package/use-onclickoutside) to close dropdowns on an outside click:

```jsx
import useOnClickOutside from "use-onclickoutside";

const DropdownDemo = () => {
  const ref = useRef(null);

  useOnClickOutside(ref, () => {
    if (isShown) setIsShown(false);
  });

  // ...

  return (
    <div className="dropdown-container" ref={ref}>
      {/* rest of dropdown... */}
    </div>
  );

  //...
};
```
