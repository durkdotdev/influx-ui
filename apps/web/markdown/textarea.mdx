# Text Area

Text Areas are styled [`<textarea>`](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/textarea) HTML elements.

## Example

<textarea className="textarea" placeholder="Description" />
<br />
<br />

```jsx
<textarea class="textarea" placeholder="Description" />
```
