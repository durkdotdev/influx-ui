# Link Text

Link Text provides styling for text.

## Example

<a className="link-text" href="#">Click Me</a>
<br />
<br />

```jsx
<a class="link-text" href="#">Click Me</a>
```

## Color Variants

<a className="link-text link-text-primary" href="#">Click Me</a>
<br />
<a className="link-text link-text-secondary" href="#">Click Me</a>
<br />
<a className="link-text link-text-tertiary" href="#">Click Me</a>
<br />
<br />

```jsx
<a class="link-text link-text-primary" href="#">Click Me</a>

<a class="link-text link-text-secondary" href="#">Click Me</a>

<a class="link-text link-text-tertiary" href="#">Click Me</a>
```
