# Nav

Footers are styled [`<nav>`](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/footer) HTML elements.

## Example

<footer className="footer">
  <a className="link-text" href="#">
    Influx UI
  </a>
</footer>
