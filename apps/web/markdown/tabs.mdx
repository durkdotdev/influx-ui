# Tabs

Use tabs to present different UI options to users.

## Example

<div className="tabs">
  <ul className="tabs-list">
    <li className="tab tab-active">Account</li>
    <li className="tab">Settings</li>
  </ul>
</div>
<br />

```jsx
<div className="tabs">
  <ul className="tabs-list">
    <li className="tab tab-active">Account</li>
    <li className="tab">Settings</li>
  </ul>
</div>
```
