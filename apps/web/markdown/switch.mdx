# Switch

Switchs are styled [`<input[type=checkbox]>`](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/input) HTML elements.

## Example

<input className="switch" type="checkbox" />
<br />
<br />

```jsx
<input className="switch" type="checkbox" />
```
