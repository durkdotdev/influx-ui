# Button

Buttons are styled [`<button>`](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/button) HTML elements.

## Example

<button className="button">Click Me</button>
<br />
<br />

```jsx
<button class="button">Click Me</button>
```

<br />
