# Select

Selects are styled [`<select>`](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/select) HTML elements.

<div className="select">
  <select>
    <option value="horizontal">Horizontal</option>
    <option value="vertical">Vertical</option>
  </select>
</div>
<br />

```jsx
<div className="select">
  <select>
    <option value="horizontal">Horizontal</option>
    <option value="vertical">Vertical</option>
  </select>
</div>
```
