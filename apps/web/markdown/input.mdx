# Input

Inputs are styled [`<input>`](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/input) HTML elements.

## Example

<input className="input" placeholder="First Name" />
<br />
<br />

```jsx
<input className="input" placeholder="First Name" />
```
