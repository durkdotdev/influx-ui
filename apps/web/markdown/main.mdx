# Main

Main is a slightly styled [`<main>`](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/main) HTML element.

## Example

<main className="main">Page content...</main>

```jsx
<main className="main">Page content...</main>
```

## Sizes

Sizes affect the bottom/top padding of the content within `<main>`.

<main className="main main-sm">Page content...</main>
<div className="divider" />
<main className="main">Page content...</main>
<div className="divider" />
<main className="main main-lg">Page content...</main>

```jsx
<main className="main main-sm">
  Page content...
</main>

<main className="main">
  Page content...
</main>

<main className="main main-lg">
  Page content...
</main>
```

## Full Screen Height

<main className="main main-screen">Page content...</main>

```jsx
<main className="main-fit-screen">Page content...</main>
```

## Centered Container

<main className="main main-centered">Page content...</main>

```jsx
<main className="main main-centered">Page content...</main>
```
